# 기능 요구사항

---

## 1. 입력 class
1. 방문날짜 입력
   - 숫자 에러처리
   - 1 이상, 31 이하 에러처리
   

2. 주문 메뉴와 개수 입력
   - 메뉴 파싱
   - 메뉴 형식 에러 처리
   - 메뉴 개수 에러 처리(숫자, 1이상)

## 2. 메뉴판
1. 메뉴판 enum
2. 메뉴 조회(없는 메뉴 에러 처리)
3. 음료만 주문 시 에러 처리
4. 메뉴 중복 에러 처리
5. 메뉴 개수 최대 제한(20개 이하)


## 3. 할인
1. 총주문 금액 계산 및 > 10,000 확인
2. 증정 이벤트
3. D-day 할인 (25일 까지만)
4. 평일, 주말, 특별 할인 (31일 까지)

## 4. 출력 class
1. 혜택 문구
2. 주문 메뉴
3. 할인 전 총주문 금액(주문 총액)
4. 증정 메뉴
5. 혜택 내역
6. 총혜택 금액
7. 할인 후 예상 결제 금액
8. 12월 이벤트 배지
